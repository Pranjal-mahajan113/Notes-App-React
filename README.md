# React Notes App

A minimal note-taking application built using React + Vite.

## 🚀 Features

- Create Groups with custom colors
- No duplicate group names allowed
- Minimum 2 character validation for group name
- Display picture with group initials
- Add notes inside specific groups
- Notes saved using Enter key or Send button
- Notes & groups persist using LocalStorage
- Date & time metadata stored with each note
- Responsive design (Mobile + Desktop)
- Send button changes color when text is entered

---

## 🛠 Tech Stack

- React (Hooks)
- Vite
- CSS Modules
- LocalStorage (Browser Storage)

---

## 📦 Installation

```bash
npm install
npm run dev
```

---

## 📱 Responsive Design

- Desktop: Sidebar + Notes panel visible
- Mobile: Sidebar first → Notes screen opens on group click
- Back navigation supported

---

## 🔐 Storage

Data is stored in:
- LocalStorage (groups)
- LocalStorage (notes)
